---
name: eye-painting-order
description: Correct step order for painting eyes on figures — sclera before iris
type: reference
---

Always paint the sclera (whites) before the iris in eye painting steps.

**Why:** Painting the iris first makes the sclera harder to apply cleanly around it. Sclera first gives a clean white base to cut into with iris colour — much easier brush control.

**How to apply:** In any figure painting guide's eye section, the step order must be: socket shadow → sclera → iris → iris highlight/pupil → catch-light → gloss.
