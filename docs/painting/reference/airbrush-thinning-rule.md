---
name: Airbrush thinning rule of thumb (nozzle size = paint %)
description: Global rule for airbrush thinning ratios — nozzle diameter as a decimal equals the paint fraction, remainder is thinner. Must appear as a highlighted callout in every guide's Thinning Reference tab.
type: reference
---
**Rule:** The thinner-to-paint ratio should roughly match the needle/nozzle size expressed as a percentage. The nozzle diameter as a decimal *is* the paint fraction — the remainder is thinner.

| Needle/Nozzle | Paint % | Thinner % | Approx ratio |
|---|---|---|---|
| 0.5mm | ~50% | ~50% | 1:1 paint:thinner |
| 0.3mm | ~30% | ~70% | 1:2.3 paint:thinner |
| 0.2mm | ~20% | ~80% | 1:4 paint:thinner |

**Caveats to always include:** This is a starting point — paint pigment density, brand viscosity, and PSI all affect the ideal ratio. Always test on a scrap card first. A well-thinned mix should flow off a palette knife in a slow, unbroken stream (not drip in blobs, not run like water). Spits or spatters → add thinner. Loses opacity too fast → add paint.

**How to apply:** Include this as a prominent highlighted callout box near the **top** of every guide's Thinning Reference tab, before the per-technique table. Apply to all new guides and retroactively to existing guides when they are edited.

**Exception — Vallejo Metal Color (VMC):** VMC is pre-thinned at the factory and does NOT follow the nozzle-size rule. Use VMC's own ratios (0.4mm → 1:1 paint:water; 0.3mm → 1:1.5 to 1:2; 0.5mm → slightly less than 1:1). Do not apply the nozzle-percentage formula to VMC — it will over-thin. See [[metallic-preferences]].
