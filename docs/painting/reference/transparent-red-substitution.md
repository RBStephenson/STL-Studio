name: Transparent Red → FW Crimson Ink Substitution
description: PA Transparent Red shifts magenta when heavily thinned; both it and FW Crimson Ink work well undiluted. FW Crimson Ink is the substitute only when a genuinely thinned/flowing application is needed.
type: reference
---

**Core finding (tested May 2026):** PA Transparent Red works correctly undiluted or at low dilution (up to ~1:2). It shifts magenta only when heavily thinned (1:4 or more). FW Crimson Ink (513) also works well undiluted. For underpainting steps — recesses, shadows, bendy parts — use either paint undiluted or lightly thinned; do not pre-thin to glaze consistency.

**Why:** Heavy dilution destabilizes the PA Transparent Red pigment, causing a visible shift toward magenta. The same thinning threshold applies roughly to both paints for underpainting purposes — don't thin either.

**How to apply:**

- **Underpainting and recess work (Methods 1, 2A, 2B):** Use PA Transparent Red 047 or FW Crimson Ink undiluted (or 1:2 max). Either works. Recommend whichever is in inventory.
- **Heavy-dilution glazes (1:4 or more):** Use FW Crimson Ink only. PA Transparent Red will shift magenta at this dilution.
- **Method 3 Wash Tinting "red wash" step:** Use undiluted and wipe back rather than pre-thinning. If flow is needed, thin FW Crimson Ink — not PA Transparent Red.
